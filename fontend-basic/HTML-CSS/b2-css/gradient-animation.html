CSS Gradients & Animations — Demo

Tài liệu này minh họa linear/radial/conic (kể cả dạng repeating), gradient cho text, border, mask, và các thuộc tính animation/transition đầy đủ: animation-name, duration, timing-function, delay, iteration-count, direction, fill-mode, play-state + shorthand.

1. Linear Gradients

linear-gradient()

Cú pháp cơ bản: linear-gradient( góc|hướng, màu1 [vị trí] , màu2 [vị trí], ...). Ở đây dùng góc 90deg.

Stops & Angle

Đặt color stops với phần trăm để điều khiển pha trộn màu.

repeating-linear-gradient()

Lặp lại theo mẫu để tạo sọc/họa tiết.

2. Radial Gradients

circle at x y

Cú pháp: radial-gradient(shape size at pos, ...). Ở đây là circle at 30% 30%.

ellipse & extent

Có thể dùng ellipse + kích thước tùy ý (ví dụ 120% 80%).

repeating-radial

Tạo vòng tròn lặp dạng sóng/đốm.

3. Conic Gradients

conic-gradient()

Tốt cho biểu đồ quạt: conic-gradient(from góc at tâm, màu start end, ...).

repeating-conic

Tạo pattern nan quạt/checkerboard.

4. Gradient đặc biệt

Gradient Text & Border

Text: background-clip:text + -webkit-text-fill-color:transparent.
Border: border-image: linear-gradient(...) 1.

Mask & Shimmer

Shimmer di chuyển bằng background-position trong @keyframes. Phía dưới có mask-image để làm mờ dần cạnh.

Nút dùng transition

Di chuột để xem hiệu ứng

Dùng transition cho tương tác nhanh; animate transform/opacity để mượt.

6. Ghi nhớ nhanh (cheatsheet)

  • Linear: linear-gradient( to right | 45deg, color stop... )
  • Radial: radial-gradient( circle|ellipse size at x y, ... )
  • Conic: conic-gradient( from deg at x y, color start end, ... )
  • Repeating: thay repeating-* để lặp pattern.
  • Text: gradient vào background + background-clip:text + -webkit-text-fill-color:transparent.
  • Border: border-image: linear-gradient(...) 1 hoặc mask với padding-box.
  • Mask: mask-image: linear-gradient(black → transparent) (Safari: thêm -webkit-mask-image).
  • Transition: transition: property duration timing delay.
  • Animation (shorthand): animation: name duration timing delay iter direction fill play-state.
  • Easing: ease | linear | ease-in | ease-out | ease-in-out | cubic-bezier() | steps().
  • Hiệu năng: Ưu tiên transform & opacity, dùng will-change cho phần tử tương tác.
  • Truy cập: Tôn trọng @media (prefers-reduced-motion: reduce).

💡 Mẹo: Bạn có thể điều khiển màu trong gradient qua var(--color) và animate CSS variables để đổi sắc mượt mà.
